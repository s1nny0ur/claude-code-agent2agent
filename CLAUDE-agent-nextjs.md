## TypeScript types come from the Sanity agent

All Sanity content types live in `{{DIR_A}}/sanity.types.ts` — generated by the
Sanity agent via `npx sanity typegen generate`. Do not edit it manually.

Before using a new type:

```bash
# Confirm typegen is done
cat {{BRIDGE}}/typegen-status   # must be "done"

# Confirm the type exists
grep "HeroModule" {{DIR_A}}/sanity.types.ts
```

Import types from the Sanity agent's worktree:

```typescript
import type { HeroModule, PagePayload } from '../../sanity.types'
```

Always read the schema contract before writing fetch functions:

```bash
cat {{BRIDGE}}/schema-contract.md
```

## Module registry — your critical coupling point

`{{DIR_B}}/components/modules/module-registry.ts` maps Sanity `_type` strings
to React components. Every new Sanity module needs an entry here AND a new
component file.

When notified of a new module:

1. Check the queue:
   ```bash
   cat {{BRIDGE}}/module-registry-queue.md
   ```

2. Wait for typegen:
   ```bash
   cat {{BRIDGE}}/typegen-status   # must be "done"
   ```

3. Read the type shape:
   ```bash
   cat {{BRIDGE}}/schema-contract.md
   grep "HeroModule" {{DIR_A}}/sanity.types.ts -A 20
   ```

4. Study existing module components for patterns:
   ```bash
   ls {{DIR_B}}/components/modules/templates/
   cat {{DIR_B}}/components/modules/templates/TextBlock.tsx
   ```

5. Create `{{DIR_B}}/components/modules/templates/Hero.tsx`

6. Add the registry entry in `{{DIR_B}}/components/modules/module-registry.ts`

7. Confirm back:
   ```bash
   {{BRIDGE}}/send-to-agent.sh a "Hero registered in module-registry.ts"
   ```

## Writing fetch functions

Fetch functions live in `{{DIR_B}}/lib/sanity/queries/` and call GROQ queries
defined by the Sanity agent in `{{DIR_A}}/sanity/lib/queries/`.

Before writing:

```bash
# Read the schema contract
cat {{BRIDGE}}/schema-contract.md

# Read the actual GROQ query definition
cat {{DIR_A}}/sanity/lib/queries/documents/page.ts

# Study the existing fetch pattern
cat {{DIR_B}}/lib/sanity/fetch-helpers.ts
cat {{DIR_B}}/lib/sanity/queries/getPageByPath.ts
```

## Bridge: `{{BRIDGE}}`

```bash
# Read messages from the Sanity agent

# Signal your status

```
